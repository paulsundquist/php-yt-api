How to add a channel:

Method 1 - Using Handle (Easiest):
Enter the channel's @ handle (e.g., @LinusTechTips)
The channel name will be automatically fetched from YouTube

Method 2 - Using Channel ID:
1. Go to the YouTube channel page
2. Click on any video from the channel
3. Look at the URL - the channel ID is after /channel/
Example: youtube.com/channel/UCXuqSBlHAE6Xw-yeJA0Tunw

How to find a playlist ID:
1. Go to the YouTube playlist page
2. Look at the URL - the playlist ID is after list=
Example: youtube.com/playlist?list=PLrAXtmErZgOeiKm4sgNOknGvNjby9efdf
The playlist ID is: PLrAXtmErZgOeiKm4sgNOknGvNjby9efdf

Note: Playlist IDs typically start with PL
